# HIREGENIE — Repository Initialization

Run the following commands to initialize and push the repo to GitHub:

```
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Amandevss/HIREGENIE.git
git push -u origin main
```

Or run the provided PowerShell script: `scripts/init-repo.ps1`.
